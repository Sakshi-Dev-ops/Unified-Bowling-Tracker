# PHHS Unified Bowling Tracker

A data analytics dashboard built with Python and Streamlit to track performance metrics, individual scores, and competitive growth trends for the PHHS Unified Bowling Team, ensuring an inclusive and data-driven approach to sports analytics.

## Features

🎳 **Team Performance Hub**
- Real-time team statistics and highlights
- Player score tracking and trends
- Personal milestones and achievements
- Season summary analytics

📊 **Data Management** (Coach Only)
- Add/edit player scores with dates
- Manage announcements
- Add/remove players from the roster

🔐 **Security**
- Password-protected coach panel
- View-only mode for players

## Getting Started

### Installation
```bash
pip install -r requirements.txt
```

### Running Locally
```bash
streamlit run app.py
```

### Deployment
This app is designed to run on Streamlit Cloud. Push to GitHub and connect your repository to deploy automatically.

**Default:** `bowling2025` (change in app.py for security)
